# 🌸 108 Divya Desams — The Sacred Abodes of Maha Vishnu

> சத்யம் ஜ்ஞானமநந்தம் ப்ரம்ம / **ஓம் நமோ நாராயணா** 🕉️

An interactive digital directory of the **108 Divya Desams** — the sacred Vishnu temples glorified by the twelve Alwars (Divine Poets) in the *Nalayira Divya Prabandham*.

./assets/12-alwars-cover.jpg

*The 12 Alwars whose pasurams define the sacred geography of Sri Vaishnavism.*

🌐 **Live site:** https://gurusubbaraman.github.io/DivyaDesams/

## ✨ Features

- 🗺️ **Interactive Leaflet map** with all 108 temples (color-coded by Vishnu posture)
- 📿 **Bilingual (Tamil + English + Sanskrit)** names for every deity, town, and region
- 🎼 **Complete Alwar-Pasuram matrix** — which Alwars sang about each temple with pasuram counts
- 🕉️ **Vishnu posture filter** — 🧍 Standing (60) · 🪑 Sitting (21) · 🛌 Reclining (27)
- 🚶 **Alwar journey animations** for Nammalvar (37), Thirumangai (86), Andal (13)
- 🌌 **Divine Realms panel** for celestial Divya Desams (Thirupparkadal & Paramapadam)
- 🎵 **Multiple audio sources** per temple (TTD Archive, KOYIL, YouTube)
- 📜 **Kalvettu tier classification** with direct SII PDF links
- 📍 **GPS coordinates** with Google Maps/Apple Maps/OpenStreetMap navigation
- ⭐ **Mannargudi Abhimana Kshetram** properly flagged
- 📱 **Fully mobile responsive** — bottom sheet on phones
